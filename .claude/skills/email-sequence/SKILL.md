Create a $0-email sequence following these steps:

1. **Discovery Questions**
   - What's the goal? (welcome, nurture, sales, re-engagement)
   - Who's the target audience?
   - What problem does the product/service solve?
   - What's the desired action?

2. **Sequence Structure**
   - Email 1: Welcome + value proposition
   - Email 2-N: Build trust, educate, overcome objections
   - Final: Strong CTA with urgency

3. **For Each Email**
   - Write 3 subject line variations
   - Include personalization tokens: {{first_name}}, {{company}}
   - Use PAS framework (Problem-Agitate-Solution)
   - Add clear CTA button copy
   - Suggest optimal send timing

4. **Deliverables**
   - Save to /templates/sequences/[campaign-name].md
   - Include A/B test suggestions
   - Add notes on segmentation opportunities

Check CLAUDE.md for brand voice before writing.

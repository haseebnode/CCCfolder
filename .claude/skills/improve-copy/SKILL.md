Improve the provided copy using this framework:

1. **Initial Analysis**
   - Read the original content
   - Identify the primary goal (awareness, consideration, conversion)
   - Note the target audience and stage in customer journey

2. **Evaluation Criteria**
   - **Clarity**: Is the message immediately clear?
   - **Engagement**: Does it capture attention and maintain interest?
   - **Emotion**: Does it connect emotionally with the reader?
   - **Action**: Is the CTA clear and compelling?
   - **Brand Voice**: Does it match guidelines in CLAUDE.md?

3. **Improvements**
   - Stronger headline (if applicable)
   - Clearer value proposition
   - More benefit-focused language
   - Better emotional triggers
   - Stronger CTA
   - Remove jargon and fluff

4. **Output Format**
   - **Original** (for reference)
   - **Improved Version**
   - **Key Changes Made** (bulleted list)
   - **A/B Test Suggestions** (what to test)
   - **Expected Impact** (what should improve)

Focus on conversion optimization, not just grammar corrections.
